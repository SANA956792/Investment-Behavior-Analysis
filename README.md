# 📊 Investment Behavior Analysis | Power BI Dashboard

## 📌 Project Overview
The **Investment Behavior Analysis Dashboard** is an interactive **Power BI project** designed to analyze investor behavior, preferences, and financial decision patterns.

This project explores how different factors such as **age, gender, investment duration, risk preference, and financial goals** influence investment choices.

The dashboard provides insights that can help **financial analysts, advisors, and researchers** understand investment trends and investor decision-making.

---

# 🎯 Project Objectives

- Analyze **investor preferences across different investment avenues**
- Understand **savings objectives of investors**
- Identify **age group and gender-based investment patterns**
- Study **investment duration and monitoring behavior**
- Discover **sources influencing investment decisions**
- Provide **data-driven insights for financial investment analysis**

---

# 🗂 Dataset Information

The dataset includes the following features:

| Column | Description |
|------|-------------|
| Age | Investor age |
| Gender | Male / Female |
| Investment_Avenues | Whether the investor invests |
| Mutual_Funds | Mutual fund investment rating |
| Equity_Market | Equity investment rating |
| Fixed_Deposits | Fixed deposit investment rating |
| Government_Bonds | Government bond rating |
| Duration | Investment duration |
| Invest_Monitor | Frequency of monitoring investments |
| Objective | Savings objective |
| Source | Source influencing investment |
| Expect | Expected return percentage |
